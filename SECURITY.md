# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

We take the security of DeepMine AI seriously. If you believe you have found a security vulnerability, please do NOT open a public issue. Instead, please report it privately to the maintainer.

1. Email the maintainer directly.
2. Provide a description of the issue.
3. Provide steps to reproduce.

We will acknowledge your report within 48 hours and provide a timeline for a fix.
