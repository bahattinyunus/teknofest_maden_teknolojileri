# Contributing to DeepMine AI

First off, thank you for considering contributing to DeepMine AI! It's people like you that make DeepMine AI such a great tool for the future of mining.

## Code of Conduct

By participating in this project, you are expected to uphold our [Code of Conduct](CODE_OF_CONDUCT.md).

## How Can I Contribute?

### Reporting Bugs
* Check the existing issues before opening a new one.
* Use a clear and descriptive title.
* Provide a minimal reproducible example.

### Suggesting Enhancements
* Explain why this enhancement would be useful.
* Provide specific examples of the new functionality.

### Pull Requests
1. Fork the repo and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. If you've changed APIs, update the documentation.
4. Ensure the test suite passes (ROS 2 build and colcon tests).
5. Make sure your code follows the project's style (C++: ROS 2 style, Python: PEP 8).

Refer to the [README.md](README.md) for installation and build instructions.
